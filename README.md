# tslint-config

TSLint Config for Ridi

## Installation

```shell
$ npm install @ridi/tslint-config --dev
```

## Usage

This is a sample `tslint.json` configuration:

```json
{
  "extends": ["@ridi/tslint-config"],
  "rules": {}
}
```
